You want to update your folder so that it is up to date with the latest code
	1. $git pull

You want to send all of your saved changes to git
	1. Make sure that all changes are saved
	2. $git add -A
	3. $git commit -m "[message for committing]"
	NOTE: Always have a message. If you do not then a strange screen will open in cmd. To get out of
		press ESC then type: ":wq" then press enter
	4. $git push




Other git commands:
	1. $git status
		This will show the status of the git situation
